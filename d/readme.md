## 1. '선택불공제'란 무슨 뜻인가요?
* 국세청 시스템이 가맹점 이름(PG사, 온라인 쇼핑몰 등)만 보고 사업용 지출인지 사적 지출인지 판단할 수 없어 **"사장님이 직접 공제 여부를 선택하라"**고 보류(기본값 불공제)해 둔 상태입니다.
* 실제 사업에 쓴 지출(양말 매입, 택배비 등)이 확실하다면 사장님이 직접 **[공제대상]으로 변경하여 부가세 환급(공제)**을 받아야 합니다.
